# Grace Insurance Group — One‑Page Site

A lightweight React + Tailwind site ready for GitHub Pages.

## Quick start

```bash
npm install
npm run dev
```

Edit contact info in `src/App.jsx` (PHONE, EMAIL) and footer licensing text.

## Build

```bash
npm run build
npm run preview
```

## Deploy to GitHub Pages

This repo includes a GitHub Actions workflow that builds the site and publishes `dist/` to Pages automatically.
